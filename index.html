Resent presentations with the posters and reports

Dec. 6th 2019

An Enhanced Sketch2Scene Using Natural Language

Translating a sketch into a scene is a challenging computer vision task. Realistic and meaningful scenes should contain multiple objects as well as a corresponding background, which is hard to extrapolate from a single input object. In this project, we propose a new approach to enhance the Sketch2Scene translation task using natural language processing techniques.

SFU AI Poster Session 2019 Project Report
Dec. 8th 2018

Children Stories Generator from Hand Drawings

Children sometimes express their feelings and thoughts through their drawings. It would be awesome if we can capture their drawings and interpret their own stories. For this purpose, we suggest the children stories generator from hand drawings using the convolutional neural network (CNN) and recurrent neural network (RNN).

SFU AI Poster Session 2018 Project Report
